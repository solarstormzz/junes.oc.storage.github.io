GitHub Sync Setup

Your characters will be saved to a private GitHub Gist and synced across all your devices. You only need to set this up once per browser.

Generate a token at GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) with only the gist scope ticked.

Create your Gist at gist.github.com — add a file called characters.json containing [], set it to Secret, and copy the ID from the URL.
